## Getting it up and running
In most cases:
```
git clone git://github.com/gueorgui/sanpo.git
cd sanpo
bundle install
rail s
open "http://localhost:3000"
```

You will also need to configure email delivery to match your
environment (see bottom of ```config/application.rb```).
